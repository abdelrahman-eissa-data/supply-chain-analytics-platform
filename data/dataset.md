# Data Folder

This folder contains the datasets used in the **Supply Chain Analytics Platform** project.

Due to file size limitations, the original dataset is not stored directly in this repository.

## Dataset Information

- **Dataset Name:** DataCo SMART SUPPLY CHAIN FOR BIG DATA ANALYSIS  
- **Source:** Kaggle  
- **Format:** CSV  
- **Number of Records:** ~180,000 rows  

## Description

The dataset includes transactional data related to supply chain operations, including:

- Orders
- Customers
- Products
- Markets and regions
- Sales and profit information
- Shipping and delivery performance

This dataset is used for data cleaning, exploratory data analysis, data warehouse modeling, and dashboard reporting.
