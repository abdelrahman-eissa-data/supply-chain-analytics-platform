# Dashboard Images

This folder contains screenshots of the **Power BI dashboards** developed for the Supply Chain Analytics Platform project.

These images provide a quick visual overview of the analytical results and key performance indicators presented in the dashboard.

## Available Visualizations

- **dashboard-management.png** – Management overview dashboard showing revenue, profit, and key operational KPIs.

- **dashboard-logistics.png** – Logistics performance dashboard analyzing shipping delays and delivery efficiency.

- **dashboard-product.png** – Product performance dashboard highlighting product profitability and category insights.

These screenshots are referenced in the main project README to illustrate the final business intelligence outputs.
